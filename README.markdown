# Tasky

Tasky is the easiest way to [Get Things Done](http://en.wikipedia.org/wiki/Getting_Things_Done).

## How it Works

* Get things off your mind
* Group tasks into projects
* Work in contexts
* Focus on the next action
* Review weekly

## Features

* Stupid Simple!
* No registration
* Clean design
* One-page app
* Everything you want
* Nothing you don't!
